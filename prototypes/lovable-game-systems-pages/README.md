# Lovable Game Architect Prototype

This is a standalone Lovable prototype, kept separate from the main app to avoid mixing architectures. Just a demonstration of what I was going for with the system design stuff. 

This is NOT integrated with the Django backend or the main frontend, nor should we try to migrate this to that; It is meant as a reference/prototype for UX and product direction and lovable is so easy to spin these things up with. 

## Run it

```bash
cd prototypes/lovable-game-architect
npm install
npm run dev
